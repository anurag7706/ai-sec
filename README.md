# AI Security Ultimate Field Guide

A comprehensive, interactive reference for securing AI systems. This guide covers foundational concepts, attack patterns, defense architectures, regulatory frameworks, and real-world incident analysis.

**Live Demo:** [https://anurag7706.github.io/ai-sec/](https://anurag7706.github.io/ai-sec/)

---

## Overview

This project consolidates research from security professionals, red teamers, and AI researchers into a single HTML-based reference. It is designed for developers, security engineers, and researchers who need a practical understanding of AI security threats and countermeasures.

---

## Contents

### Foundational Concepts
- Evolution of AI systems (Rule-Based → ML → Deep Learning → Foundation Models)
- LLM pipeline architecture and security implications at each stage
- Attack surface analysis across 7 stages

### Attack Patterns
Interactive simulations covering:
- Prompt Injection (Direct and Indirect)
- Jailbreak Techniques
- RAG Poisoning
- Memory and Context Attacks
- Tool/Function Calling Abuse
- MCP Exploitation

### Security Frameworks
- OWASP Top 10 for LLMs (2025 Edition)
- Arcanum Prompt Injection Taxonomy
- MCP Security Model
- RAG Security Architecture

### Defense & Implementation
- Defense-in-Depth Architecture (6 layers)
- AARM (Agentic AI Risk Model) Runtime Security
- Prompt Defense Patterns
- Implementation Checklist (Tier 1-3)
- Red Teaming Tools and Cadence
- Security Metrics and KPIs

---

## Usage

Open `index.html` in any modern web browser. No build process, dependencies, or server required.

Navigation is available via the top menu. Attack pattern sections include interactive "Start Simulation" buttons for step-by-step walkthroughs.
